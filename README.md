# 基于Python的豆瓣电影数据可视化分析系统 python449

## 项目概述

本项目是一个基于Python的大数据可视化系统，旨在对豆瓣电影数据进行综合分析与可视化展示，帮助用户深入了解电影产业趋势、影片评价与演员表现。

## 技术栈

- 前端：Echarts
- 后端：Flask
- 数据库：MySQL
- 数据分析：Pandas、NumPy
- 可视化：Matplotlib、Seaborn

## 功能模块

- 数据采集：从豆瓣电影网站获取电影数据
- 数据清洗：确保数据质量与一致性
- 数据分析：
  - 电影评分分布
  - 不同类型电影的数量分布
  - 演员影响力分析
- 可视化展示：基于Echarts和Flask的交互式图表

## 系统角色

- 用户：可以通过系统查询和过滤电影信息

## 运行环境

- Python环境
- MySQL数据库
- Flask框架

## 部署步骤

1. 配置Python环境
2. 安装依赖包
3. 配置MySQL数据库
4. 运行数据爬取脚本
5. 运行Flask服务器
